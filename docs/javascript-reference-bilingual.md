# JavaScript Reference (AR/EN)

Source: MDN JavaScript reference and ECMAScript language keywords.

## Keywords

`await`, `break`, `case`, `catch`, `class`, `const`, `continue`, `debugger`, `default`, `delete`, `do`, `else`, `export`, `extends`, `finally`, `for`, `function`, `if`, `import`, `in`, `instanceof`, `let`, `new`, `return`, `super`, `switch`, `this`, `throw`, `try`, `typeof`, `var`, `void`, `while`, `with`, `yield`

## Common APIs

- DOM: `querySelector`, `querySelectorAll`, `addEventListener`, `classList`, `dataset`
- Async: `Promise`, `fetch`, `async/await`, `AbortController`
- Arrays: `map`, `filter`, `reduce`, `find`, `some`, `every`
- Objects: `Object.keys`, `Object.values`, `Object.entries`, `Object.assign`
- Storage: `localStorage`, `sessionStorage`, `JSON.parse`, `JSON.stringify`

## Terminology (AR/EN)

- متغير = Variable
- دالة = Function
- كلاس = Class
- وعد = Promise
- كائن = Object
- مصفوفة = Array
- حدث = Event
- مستمع حدث = Event Listener
