## [0.0.9]

### تمت الإضافة

- إضافة مجموعة سنابيتات HTML ثنائية اللغة كاملة (141 وسمًا).
- إضافة مجموعة واسعة من سنابيتات خصائص CSS ثنائية اللغة (526 خاصية).
- إضافة سنابيتات JavaScript عملية ثنائية اللغة مع سنابيت مرجع سريع.
- إضافة ملفات مرجعية مولدة داخل `docs/` لـ HTML/CSS/JS.
- إضافة سكربت التوليد `scripts/generate-snippets.js` لمزامنة السنابيتات والوثائق.
- توسيع لغات مساهمة السنابيتات إلى `html` و`css` و`javascript` و`typescript`.

### تم التعديل

- مزامنة محتوى README العربي والإنجليزي مع آخر ترجمات Prefix العربية.
- إعادة توليد `docs/*.md` لتعكس Prefix الفعلية (AR/EN) من ملفات السنابيتات.
- تحديث `scripts/generate-snippets.js` للحفاظ على ترجمات Prefix العربية الموجودة.
- إضافة قسم "روابط مهمة" في ملفي README.
- إضافة عبارة الختام العربية: "إذا أعجبك المشروع، لا تنسَ إضافة ⭐!".
- تقليل حجم صورة `arabictohtml.png` وتقليل عرضها في README لتخفيف حجم الأصول.
- إعادة كتابة ملفات README وتحديث إرشادات المساهمة.
