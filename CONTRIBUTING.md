# المساهمة في ArabicToHtml

شكراً اهتمامك بالمساهمة في هذا المشروع! 🎉

## كيفية المساهمة

### إبلاغ الأخطاء
إذا وجدت خطأ، يرجى:
1. التحقق من عدم وجود issue مماثل
2. فتح issue جديد مع:
   - وصف واضح للمشكلة
   - خطوات إعادة الإنتاج
   - الإصدار المستخدم

### إضافة السنابيتات الجديدة
1. عدّل `snippets/snippets.code-snippets`
2. أضف كائن جديد بصيغة JSON صحيحة
3. استخدم أسماء عربية واضحة للـ `prefix`
4. أرسل Pull Request مع وصف التغييرات

### تحسين الوثائق
- يمكنك تحسين README أو إضافة أمثلة
- ترجمة المحتوى إلى لغات أخرى
- إضافة شروحات للسنابيتات المعقدة

## معايير الجودة

- استخدم الأسماء العربية الواضحة
- تأكد من صحة JSON في snippets
- اختبر السنابيت قبل الإرسال
- اكتب وصفاً واضحاً في commit

## عملية إرسال التحسينات

1. Fork المستودع
2. أنشئ فرع جديد: `git checkout -b feature/my-feature`
3. اعمل التغييرات
4. احفظ التغييرات: `git commit -m "feat: add new snippets"`
5. ارفع الفرع: `git push origin feature/my-feature`
6. افتح Pull Request
