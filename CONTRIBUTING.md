# المساهمة في ArabicToHtml

شكراً اهتمامك بالمساهمة في هذا المشروع! 🎉

## كيفية المساهمة

### إبلاغ الأخطاء
إذا وجدت خطأ، يرجى:
1. التحقق من عدم وجود issue مماثل
2. فتح issue جديد مع:
   - وصف واضح للمشكلة
   - خطوات إعادة الإنتاج
   - الإصدار المستخدم

### إضافة/تعديل السنابيتات
المشروع أصبح يعتمد على توليد تلقائي للسنابيتات والوثائق.

1. عدّل سكربت التوليد: `scripts/generate-snippets.js`
2. شغّل الأمر:
   - `node scripts/generate-snippets.js`
3. تأكد من تحديث الملفات الناتجة:
   - `snippets/snippets.code-snippets`
   - `snippets/snippets_en.code-snippets`
   - `docs/*.md`
4. أرسل Pull Request مع توضيح سبب الإضافة أو التعديل

### تحسين الوثائق
- يمكنك تحسين README أو إضافة أمثلة
- ترجمة المحتوى إلى لغات أخرى
- إضافة شروحات للسنابيتات المعقدة

## معايير الجودة

- استخدم الأسماء العربية الواضحة
- لا تعدّل ملفات السنابيتات الناتجة يدويًا إلا للضرورة؛ الأفضل التعديل عبر سكربت التوليد
- تأكد من صحة JSON في ملفات السنابيتات بعد التوليد
- اختبر السنابيت قبل الإرسال
- اكتب وصفاً واضحاً في commit

## عملية إرسال التحسينات

1. Fork المستودع
2. أنشئ فرع جديد: `git checkout -b feature/my-feature`
3. اعمل التغييرات
4. احفظ التغييرات: `git commit -m "feat: add new snippets"`
5. ارفع الفرع: `git push origin feature/my-feature`
6. افتح Pull Request
