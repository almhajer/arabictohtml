# 🌟 نظرة عامة

<div style="background: linear-gradient(135deg, #0f4c81 0%, #3a7bd5 100%); padding: 22px; border-radius: 12px; color: white;">

> **ArabicToHtml** هي إضافة VS Code تقدّم سنابيتات عملية ثنائية اللغة لتسريع كتابة HTML وCSS وJavaScript بدون الحاجة لتذكر الصياغة الكاملة لكل عنصر أو خاصية.

</div>

</div>

<div align="center">

| الفائدة | التفاصيل |
|:------:|:---------|
| ⚡ | كتابة أسرع عبر Prefix عربي أو إنجليزي |
| 🎯 | تركيز على الاستخدام اليومي في الواجهة الأمامية |
| 🌍 | مناسبة للمطورين العرب والفرق الثنائية اللغة |
| 🧩 | تغطية واسعة للعناصر والخصائص والسيناريوهات الشائعة |

</div>

# 🚀 أمثلة Prefix سريعة

</div>

### HTML

- `صفحة` أو `html page`
- `وسم قسم` أو `tag div`
- `وسم صورة` أو `tag img`
- `وسم رابط` أو `tag a`

### CSS

- `خاصية لون` أو `property color`
- `خاصية قالب أعمدة الشبكة` أو `property grid-template-columns`
- `خاصية خط حجم` أو `property font-size`
- `متغيرات css` أو `css vars`

### JavaScript

- `مستمع حدث` أو `js event listener`
- `جلب بيانات` أو `js fetch async`
- `مرجع جافاسكربت` أو `javascript quick reference`

# 📥 التثبيت

</div>

### من VS Code Marketplace

1. افتح VS Code.
2. افتح `Extensions`.
3. ابحث عن `ArabicToHtml`.
4. اضغط Install.

### عبر VSIX

```bash
code --install-extension arabictohtml-0.0.9.vsix
```

---

<div align="center">

# ⚙️ التطوير والتوليد

</div>

```bash
node scripts/generate-snippets.js
```

ينتج هذا الأمر:

- تحديث ملفات السنابيتات العربية والإنجليزية.
- الحفاظ على ترجمة الـPrefix العربية الموجودة في `snippets/snippets.code-snippets`.
- تحديث ملفات التوثيق داخل `docs/` اعتمادًا على الـPrefix الفعلية.
